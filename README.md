# 🎀 Hello Kitty Analog Clock

## 📖 Project Overview

**Hello Kitty Analog Clock** is a responsive web-based analog clock built using **HTML, CSS, and JavaScript**. The project combines the functionality of a real-time analog clock with a cute Hello Kitty-inspired user interface, creating an engaging and visually appealing frontend application.

The clock dynamically updates every second using JavaScript and accurately rotates the hour, minute, and second hands based on the user's local system time. The interface is designed with a soft pastel theme and responsive layout, making it suitable for desktops, tablets, and mobile devices.

This project was developed to strengthen my understanding of frontend development, DOM manipulation, CSS positioning, transformations, animations, and JavaScript time-based functions.

# ⚙️ How It Works

The application retrieves the current system time using JavaScript's `Date` object. Every second, it calculates the correct rotation angles for the hour, minute, and second hands and updates their positions using CSS transforms.

The clock automatically stays synchronized with the user's local time without requiring any manual input or page refresh.
